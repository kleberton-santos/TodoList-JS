# To Do App

## Visão Geral

Este projeto é uma aplicação de tarefas (To Do App) desenvolvida com HTML, CSS, JavaScript e Bootstrap, permitindo que o usuário adicione, visualize, edite, e remova tarefas. A aplicação também salva as tarefas no armazenamento local (local storage) do navegador, garantindo que os dados persistam entre as sessões.

## Funcionalidades

- **Cadastro de Tarefas**: Permite o registro de uma nova tarefa especificando a data, tipo e descrição.
- **Listagem de Tarefas**: Mostra todas as tarefas cadastradas, com a possibilidade de filtrá-las por data, tipo, ou descrição.
- **Edição de Tarefas**: Possibilita a edição dos detalhes de uma tarefa existente, como data, tipo, e descrição.
- **Conclusão de Tarefas**: O usuário pode marcar uma tarefa como concluída ou desmarcá-la.
- **Remoção de Tarefas**: Permite excluir uma tarefa da lista e do local storage.
- **Filtro de Tarefas**: Facilita a busca de tarefas específicas com base em critérios como ano, mês, dia, tipo e descrição.

## Tecnologias Utilizadas

- **HTML, CSS e Bootstrap** para estruturação e estilização da interface.
- **JavaScript** para a lógica de manipulação das tarefas, incluindo a integração com o local storage.
- **Local Storage** para persistência de dados no navegador.

## Estrutura do Projeto

O projeto é composto pelos seguintes arquivos:

- `index.html`: Página principal onde o usuário pode cadastrar tarefas.
- `listTask.html`: Página onde o usuário visualiza, edita e remove tarefas.
- `app.js`: Arquivo JavaScript que contém toda a lógica da aplicação.
- `style.css`: Arquivo de estilos adicionais para a aplicação.
